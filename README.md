# T Budget

Application de budget mensuel — fond beige, police Georgia.

## Déploiement sur Vercel

1. Importer ce repository sur vercel.com ("Add New Project" → sélectionner ce repo)
2. Vercel détecte automatiquement Vite/React — laisser les réglages par défaut
3. Cliquer "Deploy"
4. Ouvrir le lien fourni sur iPhone (Safari) → icône de partage → "Sur l'écran d'accueil"
